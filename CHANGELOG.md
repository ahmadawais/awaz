Initial release.

- `awaz voices` — list available voices
- `awaz prompting` — tips for better output
- Streaming by default
- Mac `say` compatible flags (`-v`, `-r`, `-o`, `-f`)
- Multiple models: v3, v2, Flash, Turbo
